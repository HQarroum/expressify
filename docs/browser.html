Select an URL and a resource for your request.

Use the example

If you do not know of any Expressify online resource, or you just want to try the browser, we provide an Expressify server application that saves an item you provide using a JSON payload in its local storage, and returns it on demand.

Its application URL is https://hqarroum.github.io/expressify/server.html, and you can call the resource /item/:id with a GET request to retrieve an item having a given id, using a POST request to store a new item, and DELETE to remove the resource. Similarly, as with any Expressify server, you can ask it to describe the RESTful resources it currently exposes, along with their description.

Another resource made available through the example server is the /echo resource which returns back an object holding the request of the client.
